Edit Menu

Copy Board Image to Clipboard

Places a bitmap image of the selected board onto the clipboard.

Save Board Image in File

Stores a bitmap image of the selected board in a BMP format disk file.

Delete Selected Markers

Destroys all selected markers.

Select All Markers

Selects all markers on the current board view.

Select All Markers From

Selects all objects that were created from a specific marker group. For instance, this could be is useful for deleting all combat markers.

Edit Object Text

Displays a dialog used to edit an object's associated text. Only a single object's text may be edited at a time. You may also double-click an object to use this dialog.

Board Properties

Displays the properties dialog for the current playing board.
