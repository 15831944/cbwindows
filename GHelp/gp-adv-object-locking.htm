How Object Locking Works

Pieces and markers can be locked to the board's surface. When an object is locked it is impossible to select it. It's as if the object is part of the original board design.

Objects can be locked while in either Scenario or Game mode.

To use the locking feature first select the objects you wish to lock. Then use the Actions/Lock Objects menu command. The objects will be locked and deselected.

If you need to unlock a piece you would first deactivate the locks with the Actions/Suspend Object Locks menu command. This gives you access to all locked object on the board. Then select the objects you wish to unlock and use the Actions/Lock Objects to unlock them. Finally reactivate the locks with the "Actions/Suspend Object Locks" menu command.

Some possible applications of this feature are:

  • Customizing a board's terrain prior to play.
  • Building tile based maps.
  • Build RoboRally like boards from smaller map pieces.
  • Locking down pieces you don't accidentally want moved.
  • Adding board annotations during play that you don't want to be selected.
