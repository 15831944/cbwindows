How to display a different board bitmap at each board viewing scale.

Use the Tools/Set Scale Visibility feature to do this. The procedure in more detail:
  1. Create the full scale map and use the CyberBoard palette to create a compatible image. Paste this into the playing board.
  2. Open the Set Scale Visibility dialog and clear all scale check marks except full-scale. You will now only see this bitmap when the board is viewed at full-scale.
  3. Open the board properties dialog and make note of the pixel dimensions of the half and small scale boards. These are the target sizes you should use when creating your reduced scale maps.
  4. Set the board scale to half-scale and paste the half-scale bitmap into the playing board. Note: it will initially seem too small.
  5. Open the Set Scale Visibility dialog and check the "Above scale is natural scale" and "Half Scale" check boxes.
  6. Set the board scale to small-scale and paste the small-scale bitmap into the playing board. Note: it will initially seem too small.
  7. Open the Set Scale Visibility dialog and check the "Above scale is natural scale" and "Small Scale" check boxes.
